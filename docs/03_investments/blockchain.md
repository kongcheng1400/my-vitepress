# 考量
1. 用户转换成本.
2. 平台效应(生态及价值链)
3. 网络效应: 节点达到所有交易所/国家与地区
4. 开发及创新活动(原创生态)， 原创生态的组合性
5. 高利润及高项目/价值捕获能力.
## storage
- ipfs:没有链， 有一个网络。retrieve 快速.(没有激励，数据持久性差.)
- filecoin:链下存储链上证明. 有存储激励，数据持久性好。(矿工, 冷存储，检索慢)
- arweave: 链上存储(持久性好)，节点同步, 数据检索快. 缺点: tx size 大，同步时间长.
